什么是 Hexo？

Hexo 是一个快速、简洁且高效的博客框架。 Hexo 使用 Markdown（或其他标记语言）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装 Hexo

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

在<folder>目录下将会产生如下文件

_config.yml 是一个配置文件，用来设置 Hexo 的基本参数。

package.json 是一个 Node.js 项目配置文件，用来管理 Hexo 及其插件的依赖。

scaffolds 目录存放着新建文章的模板文件。

source 目录存放着文章的 Markdown 文件。 是存放用户资源的地方。 除 _posts 文件夹之外，开头命名为 _ (下划线)的文件 / 文件夹和隐藏的文件将会被忽略。 Markdown 和 HTML 文件会被解析并放到 public 文件夹，而其他文件会被拷贝过去。

themes 目录存放着主题文件。

配置

您可以在 _config.yml文件中修改大部分的配置。

网站

设置 描述
title 网站标题
subtitle 网站副标题
description 网站描述
keywords 网站关键词
author 作者名称
language 网站默认语言，采用ISO 639-1 编码
timezone 网站时区，默认为电脑时区，参考 Timezone，例：Asia/Shanghai

网址

设置 描述 默认值
url 网址，以http://或https://开头
root 网站根目录 url’s pathname
permalink 文章永久链接格式 :year/:month/:day/:title/
permalink_defaults 默认的永久链接参数
pretty_urls 美化URL false
